# CSS Flexbox

## Concepts
* flex container
* flex item
* axis
  - main axis
  - cross axis
* lines
  - start line
  - end line

---

## Flex Container Properties

* display
* align-content (spacing in cross axis)
* align-items (cross axis)
* justify-content (main axis)
* flex-flow
  - flex-direction
  - flex-wrap

---

## Flex Item Properties
* align-self
* flex
  - flex-grow
  - flex-shrink
  - flex-basis
* order
