# CSS Flexbox

## Concepts
* flex container
* flex item
* axis
  - main axis
  - cross axis
* lines
  - start line
  - end line

---

## Flex Container Properties

* display
* flex-flow
  - flex-direction
  - flex-wrap
* align (this is not a property)
  - align-content (spacing in cross axis)
  - align-items (cross axis)
* justify-content (main axis)

---

## Flex Item Properties
* flex
  - flex-grow
  - flex-shrink
  - flex-basis
* order
* align-self
