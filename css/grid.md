# CSS Grid

Remember that:

* **align**: top => bottom, longitude, y-axis
* **justify**: left => right, latitude, or x-axis

---

## Concepts
* grid container
* grid item
* grid line
* grid track (explicit vs. implicit)
* grid cell
* grid area
